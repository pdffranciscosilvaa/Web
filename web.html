Conceitos da Web

HTML (HyperText Markup Language)

É a linguagem de marcação usada para criar e estruturar o conteúdo das páginas web. Ele permite a inserção de texto, imagens, links, vídeos, e outros elementos multimídia.

HTTP (HyperText Transfer Protocol)

É o protocolo de comunicação usado para transferir dados na Web. Quando você digita um endereço web no seu navegador, uma solicitação HTTP é enviada ao servidor onde o site está hospedado, e o servidor responde com os dados da página.

URLs (Uniform Resource Locators)

São os endereços que você digita no navegador para acessar um site. Eles indicam a localização de recursos específicos na Web.

Navegadores

São os programas usados para acessar e visualizar páginas web. Exemplos incluem:

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

CSS (Cascading Style Sheets)

É uma linguagem usada para descrever a apresentação de um documento HTML. Ele controla o layout, cores, fontes, e outros aspectos visuais das páginas web.

JavaScript

É uma linguagem de programação que permite a criação de conteúdo interativo e dinâmico nas páginas web. É amplamente utilizada para melhorar a experiência do usuário.

Servidores Web

São computadores que armazenam e servem páginas web aos usuários. Quando um navegador faz uma solicitação para uma página, o servidor web processa a solicitação e envia a página de volta ao navegador.

HTTP/HTTPS

HTTP é o protocolo básico para comunicação na Web, enquanto HTTPS (HTTP Secure) é a versão segura, que criptografa os dados transmitidos entre o navegador e o servidor para proteger a privacidade e a integridade dos dados.

A Web foi inventada por Tim Berners-Lee em 1989 enquanto ele trabalhava no CERN, e desde então tem evoluído constantemente, incorporando novas tecnologias e padrões para melhorar a funcionalidade, acessibilidade e segurança dos recursos online.
